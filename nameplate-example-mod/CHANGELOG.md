# Changelog

All notable changes to NameplateBuilder Example Mod will be documented in this file.

## [1.0.1] - 2026-02-17

### Changed
- Updated target server version to 2026.02.17-255364b8e

## [1.0.0] - 2025-02-11

### Added
- Initial release of example mod demonstrating NameplateBuilder API usage
- `ArchaeopteryxNameplateSystem` showing NPC nameplate registration on spawn
- `LifetimeNameplateSystem` demonstrating per-tick nameplate updates
- Example segment: "Lifetime" showing entity age in ticks
- Example segment metadata with display name, target, example text, and author
- Working reference implementation for mod developers
